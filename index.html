Installation

Prise en Main & Installation Rapide

  1. 1
    Téléchargez le fichier de l'extension au format ZIP en cliquant sur le bouton ci-dessous.
  2. 2
    Ouvrez votre navigateur et accédez à la page chrome://extensions/.
  3. 3
    Activez le Mode développeur situé en haut à droite.
  4. 4
    Cliquez sur Charger l'extension non empaquetée en haut à gauche et sélectionnez le dossier de l'extension extrait.
Télécharger pour Chrome (ZIP)
  1. 1
    Téléchargez le fichier de l'extension pour Firefox.
  2. 2
    Saisissez about:debugging dans votre barre d'adresse Firefox et cliquez sur Ce Firefox.
  3. 3
    Cliquez sur Charger un module temporaire... et sélectionnez le fichier manifest.json de l'extension.
Télécharger pour Firefox
  1. 1
    Rendez-vous sur le dépôt officiel de la suite RACYWAMA sur GitHub.
  2. 2
    Clonez le dépôt en ligne de commande : git clone https://github.com/....
  3. 3
    Suivez le guide de contribution si vous souhaitez ajouter vos propres thèmes ou modules !
